## Source code

```
#include <iostream> 
using namespace std;

int main(int argc, char* argv[]) { 
	cout << "argc = " << argc << endl; 
	cout << "argv[0] = " << argv[0] << endl << endl; 

	for(int i = 1; i < argc; i++){
	cout << "argv[" << i << "] = " << argv[i] << endl; 
	}
	// return 0;  
	system("pause"); 
}
```

- `argc`   : argv[] size，吃進的參數的個數。
- `argv[]` : 吃進參數會放到 argv Array 裡面。
- `argv[0]`: 其中 index = 0 為程式的路徑。

## 執行
### Linux Command line
```
$ ./run 0 1 "aa"

=> argc = 4
=> argv[0] = ./run

=> argv[1] = 0
=> argv[2] = 1
=> argv[3] = aa
```
